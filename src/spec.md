# Better Commit Landing Page

## Overview
A simple, clean landing page that displays "Better Commit" as the main heading, centered on the page with a modern, minimal design and a solid red background.

## Features
- Display "Better Commit" text prominently centered on the page
- Solid red background color across the entire page
- Text color optimized for readability against the red background
- Clean, minimal design aesthetic
- Modern typography
- Responsive layout that works on both desktop and mobile devices
- English language content

## Technical Requirements
- Static landing page (no backend data storage required)
- Solid red background color applied to the entire page
- Text styling that ensures high contrast and readability against red background
- Responsive design for multiple screen sizes
- Modern font selection for clean appearance
- Centered layout with appropriate spacing and typography hierarchy
